# Project Gutenberg Document Loader for Langchain Community

## Description

This module provides a document loader for Project Gutenberg eBooks,
integrating with the Langchain Community's document loading system.
Project Gutenberg is an online library of free eBooks:
[Project Gutenberg](https://www.gutenberg.org/about/)

## Document Loader

See a [usage example](https://langchain-community.github.io/documentation/integrations/document_loaders/gutenberg)
in the Langchain Community documentation.




This loader fetches eBook content from Project Gutenberg using the given
book ID and returns a list of Document objects containing the text.
