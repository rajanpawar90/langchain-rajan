# Iugu Payment Processing

Iugu is a Brazilian company that provides payment-processing software and APIs for e-commerce websites and mobile applications. You can learn more about Iugu at [their website](https://www.iugu.com/).

## Installation and Setup

To use the Iugu API, you will need to obtain an access token from your Iugu dashboard.

## Document Loader

Here is an example of how to use the Iugu document loader:




The `IuguLoader` class is a part of the `langchain_community.document_loaders` module and can be used to load documents from Iugu. Simply create an instance of the `IuguLoader` class and call the `load` method, passing in the path to your Iugu document.
